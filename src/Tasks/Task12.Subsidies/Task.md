# Задача 12: Subsidies

бъявите структуру Parent, которая описывает родителя. В данной структуре надо предусмотреть следующие поля:

- полное имя
- дата рождения
- пол
- возраст (целый тип)
- идентификационный номер (серия и номер паспорта) 
- дети (массив структур Child)
- количество детей
Напишите функцию на вход которой подается массив родителей и сумма суммарной денежной субсидии (тип int) total.

Функция должна возвращать новый массив типа double, каждая ячейка которого должна содержать величину субсидии для детей этого родителя. Субсидия вычисляется следующим образом:

- субсидия полагается только детям которые не старше 16 лет и если у родителя количество детей не менее 2 детей.

- для родителей у которых количество детей 2 или более сумма субсидии для каждого ребенка c номером index вычисляется по формуле base * (1 + index * 0.1). base - это базовая ставка, которую нужно посчитать, index - номер ребенка, с нумерацией от 0.

- суммарно по всем родителям общая сумма субсидии должна быть равна total. 

- предположить, что во входном массиве есть хотя бы один родитель, которому полагается субсидия.
