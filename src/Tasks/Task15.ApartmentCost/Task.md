# Задача 15: ApartmentCost

Объявите структуру, которая бы описывала городскую квартиру с точки зрения агентства недвижимости. Предусмотрите в этой структуре возможность вводить данные о площади каждой комнаты отдельно, кухни и площади остальных помещений. Сумма этих чисел будет равняться общей площади квартиры.

Напишите подпрограмму, которой подается на вход список (объект List) с информацией о квартирах. Подпрограмма должна вычислять медиану цены для квартир у которых количество комнат равно Rooms, а общая площадь больше TotalSquare. Rooms и TotalSquare - входные аргументы функции.

https://ru.wikipedia.org/wiki/Медиана_(статистика)

Оцениваться будет грамотное объявление структуры и сам алгоритм.
