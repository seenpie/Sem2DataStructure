# Задача 13: SpaceFlight

В одномерном неотсортированном массиве типа double введено финансовое состояние (вещественное число) желающих оказаться на космическом корабле, стартующем на другие планеты. Необходимо разработать функцию, которая из массива отбирает n претендентов на космический полет по принципу - летят самые богатые.

Известно, что все элементы массива положительные числа, и что n заведомо меньше размера массива.

Постарайтесь не сортировать массив.

Вход:

total - целое, положительное число, меньше или равное size - количество претендентов, которых нужно отобрать для полета.

size - целое, положительное число не больше 1000, размер входного массива.

n1, n2... - size вещественных чисел, финансовое состояние претендентов. Не отсортированы.

Выход:

m1, m2... - числа, отобранные нашим алгоритмом в порядке невозрастания. Вывод сделать каждого числа на отдельной строке
