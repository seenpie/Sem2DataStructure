# Задача 11: SegmentsSort

Вам необходимо объявить структуру, которая описывает отрезок на плоскости (координаты, цвет линии и толщина линии).

Далее, напишите функцию, которой подается на вход одномерный массив структур (как объявлено выше). Функция должна сортировать этот массив по длине отрезков методом быстрой сортировки.
